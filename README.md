## Keymaps

| keymap                            | action                 |
| --------------------------------- | ---------------------- |
| Alt+Shift+F                       | Open                   |
| Alt+Shift+J / Tab / ArrowDown     | Select item below      |
| Alt+Shift+K / Shift+Tab / ArrowUp | Select item above      |
| Alt+Shift+D                       | Duplicate selected tab |
| Alt+Shift+W                       | Close selected tab     |
| Enter                             | Go to selected tab     |
| Alt+Shift+F / Esc                 | Close                  |
